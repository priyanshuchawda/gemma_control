Privacy isn't a setting in GemmaControl — it's a hard architectural constraint baked into every layer.

  • No cloud inference — model runs locally via LiteRT-LM
  • INTERNET permission only for model binary download

Setup guide

Up and running
in 5 steps.

Android 16 required. Tested on Xiaomi Redmi 13 5G.

  1. 01
    Download the APK & model Grab GemmaControl-v1.0.0-debug.apk and the mobile_actions_q8_ekv1024.litertlm model file from the download section above.
    Get files ↓
  2. 02
    Enable Unknown Sources Settings → Apps → Special app access → Install unknown apps → enable for your file manager or browser, for this install only.
  3. 03
    Grant Notification Access Settings → Notifications → Notification access → enable GemmaControl. This is how the app reads WhatsApp alerts.
  4. 04
    Xiaomi / HyperOS: disable restrictions Enable Autostart and set battery mode to No restrictions so the notification listener stays alive in background.
  5. 05
    Load the FunctionGemma model Open app → Settings → tap "Load model file" → select the .litertlm binary. SHA-256 is verified before LiteRT-LM initialises.
