Privacy boundary

No silent sends

Setup

Install checklist.

  1. Download the signed APK. Use the Android package from this site or the latest GitHub release.
  2. Install on Android. Allow installation from your browser or file manager only for this install.
  3. Grant notification access. GemmaControl reads visible WhatsApp notifications after explicit permission.
  4. Configure Xiaomi background settings. On HyperOS/MIUI, enable Autostart and set battery usage to no restrictions.
  5. Install the FunctionGemma model in Settings. The app verifies model downloads with SHA-256 before using LiteRT-LM.
