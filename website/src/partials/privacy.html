Privacy by design

Your data never
leaves your phone.

Privacy isn't a setting in GemmaControl — it's a hard architectural constraint baked into every layer.

  • No cloud inference — model runs locally via LiteRT-LM
  • INTERNET permission only for model binary download
  • AES-GCM 256 for all stored message metadata
  • Manual tool call execution — AI proposes, never auto-runs
  • Open source — every line auditable on GitHub
  • Zero analytics, zero telemetry, zero tracking
